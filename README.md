# 🎙️ AI Podcast Generator

An AI-powered podcast generation platform that creates podcast scripts using Google Gemini and converts them into realistic speech using Murf AI.

## 🚀 Live Demo

**Frontend:** https://ai-podcast-generator-n8n.vercel.app/

**Backend API:** https://ai-podcast-backend-production.up.railway.app

---

## ✨ Features

- 🎤 Generate podcasts from any topic
- 🤖 AI-generated podcast scripts using Google Gemini
- 🔊 Natural text-to-speech conversion using Murf AI
- 🎧 Built-in audio player
- ⚡ Fast and responsive user interface
- ☁️ Fully deployed online

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Anvitha-av/ai-podcast-generator-n8n.git
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```
