# 🎙️ AI Podcast Generator

An AI-powered podcast generation platform that converts user topics into narrated podcasts using Generative AI and workflow automation.

## 🚀 Features
- Topic-based podcast generation
- AI script generation using Google Gemini
- Text-to-speech conversion using Murf AI
- Workflow automation using n8n
- Frontend built using Lovable
- Production webhook deployment using n8n Cloud

## 🛠️ Tech Stack
- Lovable
- n8n Cloud
- Google Gemini API
- Murf AI API
- Webhooks
- REST APIs

## 🔄 Workflow

User enters topic
↓
Lovable Frontend
↓
n8n Webhook
↓
Google Gemini generates script
↓
Murf AI converts script to speech
↓
Audio returned to user
